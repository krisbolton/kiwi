Anyone can learn InfoSec skills.

Hello there,

All of the courses and modules available on Security Kiwi are learner-friendly; they use plain language, code snippets, graphics, and video which appeal to the range of learning styles. In addition, I make myself available to answer questions via GitHub Discussions and encourage the community to assist eachother.

“Security Kiwi provides people the oppurtunity to learn skills in interesting areas such as computer science, programming, and machine learning. ”

I imagine its polite to introduce myself here. Hi, I'm Kris Bolton. Security Kiwi was created as the result of my undergraduate and graduate education in computer science and cyber security and the lack of information available outside of academia and the private sector on machine learning for security.

Security Kiwi now contains much more content than just machine learning material and has grown to include content to encourage new people into information security and provides people with the oppurtunity to learn intermediate and advanced skills in interesting areas such as computer science, programming, and machine learning. I plan to expand this content to include areas such as information security policy, standards, and risk.

The Vision

The vision for Security Kiwi is a high-quality resource for those interested in learning about information security and obtaining new skills. I have and will continue to put a great deal of time and resources into Security Kiwi to make it an excellent resource. Security Kiwi is now a fully-fledged Flask web application capable of providing a significantly better learner experience than the previous static website, and continues to improve over time with new feautres and content.
