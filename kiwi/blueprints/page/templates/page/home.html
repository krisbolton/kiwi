What is Security Kiwi?

Security Kiwi is a collection of educational material for students to learn various topics in information secuirty. Its purpose is to give learners the knowledge and skills to do interesting things and apply them to their own problems and ideas. It's an oppurtunity for me (Kris Bolton) to give back to the community and continue to learn myself (you have to know a subject to teach it well!). You can read the community page to learn more.

What is GitHub Sponsors?

GitHub Sponsors is a feature built into GitHub allowing users to sponsor the creator of a project to financially support the project. Everything is handled by GitHub using your GitHub user so it's secure and easy to use. 100% of the amount goes to the creator.
